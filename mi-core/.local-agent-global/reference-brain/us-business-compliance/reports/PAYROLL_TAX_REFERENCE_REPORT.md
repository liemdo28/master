# Payroll Tax Reference Report

_Generated: 2026-06-09T12:27:20Z_

## Overview

Payroll and tax reference documents cover federal, Texas, and California payroll tax requirements for restaurant employers including deposit schedules, filing requirements, tip reporting, and compliance checklists.

## Key Topics

- **Federal Payroll**: Form 941 (quarterly), Form 940 (annual), W-2/W-3 (annual)
- **Social Security**: 6.2% each (EE/ER), $176,100 wage base (2025)
- **Medicare**: 1.45% each (EE/ER), no wage base limit
- **Additional Medicare**: 0.9% employee only over $200,000
- **Tip Reporting**: Form 8027 required for restaurants with >10 employees
- **California Payroll**: DE 9 (quarterly), SDI 1.1% employee, ETT 0.1% employer
- **Texas Payroll**: TX Submit (quarterly), SUTA on $9,000 wage base
- **FICA Tip Credit**: Form 8846, credit for employer FICA on tips above minimum wage

## Filing Calendar

- April 30: Q1 Form 941 and state returns
- July 31: Q2 Form 941 and state returns
- October 31: Q3 Form 941 and state returns
- January 31: Q4 Form 941, annual Form 940, W-2s, W-3s, 1099s
