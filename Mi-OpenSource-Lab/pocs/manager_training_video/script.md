# Manager Dashboard Training

Duration target: 90 seconds
Audience: Manager
Language: English-first, Vietnamese optional

## Sections

1. Login to the manager dashboard.
2. View assigned tasks.
3. Review a submitted task.
4. Add a comment.
5. Approve or reject.
6. Check task history.
7. Escalate an issue.

## Script

Welcome to Manager Dashboard Training. In this walkthrough, managers learn how to review daily tasks, leave clear comments, approve or reject work, and escalate issues when something needs attention.

First, log in with your manager account. Open the assigned tasks panel and review the task list. Select one submitted task, check the evidence, then add a concise comment. If the submission is correct, approve it. If something is missing, reject it with a clear reason. Finally, open the history panel to confirm the action was recorded, and use escalation when a task needs owner review.
