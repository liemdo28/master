# Mi Video Agent Spec

Provider candidate: OpenMontage

Goal: generate training-video artifacts from role, script, screenshots, voice, and subtitle requirements.

Required outputs:
- `script.md`
- `shot_list.md`
- `voiceover.txt`
- `subtitle.srt`
- `video_plan.json`
- optional `final_video.mp4`

Production rule: generated media must be stored as traceable Mi artifacts and approved before publishing.
