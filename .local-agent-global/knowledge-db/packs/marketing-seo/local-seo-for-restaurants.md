# Local SEO for Restaurants

## Google My Business (GMB)
- Complete all fields
- Add photos weekly
- Respond to all reviews
- Post updates weekly
- Enable messaging

## On-Page SEO
- Title tag: "Bakudan Ramen | Best Ramen in [City]"
- Meta description with location + key dish
- H1 must include primary keyword
- Schema markup: Restaurant, Menu, Review

## Local Citations
- Yelp, TripAdvisor, OpenTable
- Consistent NAP (Name, Address, Phone)

## Content Strategy
- Blog about local food scene
- Chef profiles
- Dish stories / ingredient sourcing
