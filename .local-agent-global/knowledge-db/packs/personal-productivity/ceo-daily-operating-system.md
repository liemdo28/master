# CEO Daily Operating System

## Morning Routine
1. Review yesterday's outcomes (5 min)
2. Identify Top 3 priorities for today
3. Block deep work time (2+ hours)
4. Review calendar for meetings

## Priority Matrix
- Urgent + Important → Do now
- Not urgent + Important → Schedule
- Urgent + Not important → Delegate
- Not urgent + Not important → Eliminate

## Weekly Review
- Review all projects
- Clear inbox to zero
- Update task list
- Plan next week

## Deep Work Principles
- No meetings before 11am if possible
- Turn off notifications during focus blocks
- Single task — no context switching
- Break every 90 minutes
