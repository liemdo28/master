# HEALTH BRIEFING INTEGRATION
**MI Health V1 — Phase 6 Deliverable**  
**Date:** 2026-06-13  
**File:** `mi-core/health/HealthBriefingIntegration.mjs`

---

## Integration Point

The morning Executive Briefing (generated by `DailySnapshotBuilder.mjs` or equivalent) calls:

```javascript
import { injectHealthIntoBriefing } from '../health/HealthBriefingIntegration.mjs';

// Inject into existing briefing object
const briefing = await buildExecutiveBriefing();
const withHealth = injectHealthIntoBriefing(briefing);
```

Or standalone:
```javascript
import { buildHealthBriefingBlock, formatHealthBriefingText } from '../health/HealthBriefingIntegration.mjs';
const block = buildHealthBriefingBlock();
console.log(formatHealthBriefingText(block));
```

## API

```javascript
buildHealthBriefingBlock(date?)
// Returns structured object:
{
  section: 'Health',
  date: 'YYYY-MM-DD',
  score: { value: 81, grade: 'B+' },
  score_delta: 3,
  score_delta_label: '+3 vs last week',
  metrics: {
    sleep:       { label, value, grade, detail },
    steps:       { label, value, goal },
    resting_hr:  { label, value },
    hrv:         { label, value },
    active_cals: { label, value }
  },
  alerts: [ { severity, message }, ... ],
  recommendations: [ 'string', ... ]
}

formatHealthBriefingText(block)
// Returns plain text for briefing output

injectHealthIntoBriefing(briefing, date?)
// Accepts string or object, returns same type with health injected
```

## DailySnapshotBuilder Integration

Add to `DailySnapshotBuilder.mjs`:

```javascript
import { buildHealthBriefingBlock } from '../health/HealthBriefingIntegration.mjs';

// In buildSnapshot():
const healthBlock = buildHealthBriefingBlock();
snapshot.health = healthBlock;
```

This makes health data appear in the daily snapshot JSON that Mi reads from.
