**Narration:** "After signing in, you land on the Dashboard. This is a quick overview of your entire Link Hub. The orange 'Action Required' bar lists real issues — draft pages, missing SEO, or unpublished changes. If it's empty, everything is in good shape. The KPI tiles show live numbers: Total Buttons, Live, Hidden, Scheduled, Views and Clicks in the last 24 hours. Quick Actions give you one-click shortcuts to every main section. And Pages Overview shows the status of each page — LIVE or HIDDEN — with a quick Edit Buttons button."
**Subtitles:**

## 2. Pages (3–4 min)
**Screen:** Click **Pages** in the sidebar.
**Narration:** "This is the control center for every page in your Link Hub. Currently you have a public Customer Link Hub and a private Staff Training page. The overview tiles show page counts by type and status. The table shows every page with its type badge — green for Customer Link Hub, purple for Staff Training — so you always know which is public and which is private. Click Edit on any page to open the page editor."
**Subtitles:**
- Control center for every page in your Link Hub.
- Green badge = public Customer Link Hub. Purple badge = private Staff Training.
- Click Edit to open the page editor.

### 2a. Page Editor — Buttons Tab
**Screen:** Buttons tab (default).
**Narration:** "The Buttons tab is where you manage every button on this page. The status bar shows the current status — Draft or Published — with buttons to Save, Publish or Unpublish, Verify, Save as Template, and go Back. The 'Save as Template' button is powerful: build a page you're happy with, save the entire structure, and reuse it in one click for a new store instead of starting from scratch. Each button row shows the title, a badge for Featured or section name, the destination URL, and toggles for Visible, Enabled, and Featured. Action icons let you Edit, Duplicate, Move to another page, or Delete. You can drag and drop to reorder buttons — a hint appears when you start dragging."
**Subtitles:**
- Manage every button on this page from the Buttons tab.
- Save as Template: build once, reuse for new stores in one click.
- Each button has Visible, Enabled, and Featured toggles.
- Drag and drop to reorder buttons.

### 2b. Button Editor — Destination Types
**Narration:** "Each button has a Destination Type — this tells the system what kind of link it is and prevents the common mistake of a broken URL. Scroll through the dropdown to see all types. Classic types include External Website, YouTube Video, Phone Number, Email, Google Maps, PDF, Toast Online Ordering, Instagram, and Facebook — each with a format hint. 'Internal Link Hub Page' links to another page you manage by name — if you change that page's URL slug, this link still works automatically. The newest types are 'Call This Location,' 'Get Directions,' 'Store Hours,' and 'Order Support Email.' Instead of typing a phone number or address manually, you pick a Location from a dropdown and the button uses that location's info automatically. Update the Stone Oak phone number in one place, and every 'Call This Location' button pointing to Stone Oak updates instantly — no need to fix five buttons separately. 'Heading,' 'Text Block,' and 'Image' are non-link blocks for section titles and text. You can also set a Start Date and End Date to schedule when a button appears or disappears automatically. Four toggles at the bottom control Visible, Enabled, Featured, and New Tab."
**Subtitles:**
- Destination Type tells the system what kind of link this is.
- Internal Link Hub Page: links survive URL slug changes automatically.
- Call This Location: pick a location, and the button auto-uses its phone, map, hours, or email.
- Set Start and End dates to schedule when a button appears or disappears.

### 2c. Page Editor — Sections Tab
**Screen:** Click **Sections** tab.
**Narration:** "Sections group buttons together — like 'Order Online' or 'Rewards & Loyalty' headers on the public page. You can create, rename, reorder, hide, schedule, or delete sections here. Every button belongs to exactly one section — or no section at all."
**Subtitles:**
- Sections group buttons together — like 'Order Online' or 'Rewards & Loyalty' headers.
- Every button belongs to exactly one section — or no section at all.

### 2d. Page Editor — Page Settings Tab
**Screen:** Click **Page Settings** tab.
**Narration:** "This tab controls the page itself — not its buttons. Set the Page Title, URL slug, Page Type, and Visibility: Public, Unlisted, Staff Only, Password Protected, or Inactive. Toggle whether it shows on the Customer Link Hub and whether search engines index it. Fill in the Profile Handle, Headline, and Tagline. For SEO, the Google preview box updates live as you type your SEO Title and Meta Description — both have character counters so you know if they're the right length. Upload an Open Graph image for social sharing, and set a Canonical URL to prevent duplicate content issues."
**Subtitles:**
- Page Title, URL slug, Page Type, and Visibility.
- Google preview updates live as you type SEO title and meta description.
- Set canonical URL to prevent duplicate content.

### 2e. Page Editor — Publish & Preview Tab
**Screen:** Click **Publish & Preview** tab.
**Narration:** "The Publish & Preview tab controls the publishing lifecycle. Choose a status: Draft, Private, Scheduled, or Published. If you choose Scheduled, a date and time picker appears to auto-publish at a specific moment. The Private Preview Link section generates a shareable link for your team to review before going live — you can regenerate the token to invalidate the old link at any time. Version History lets you load previous published versions, preview them, or restore if needed. Before publishing, the system may warn about broken links, duplicate buttons, or staff content on a public page — fix or confirm before proceeding."
**Subtitles:**
- Choose Draft, Private, Scheduled, or Published status.
- Private Preview Link: share with your team before going live.
- Version History: preview or restore any previous version.
- The system warns before publishing broken links or staff content on public pages.

## 3. Templates (1 min)
**Screen:** Click **Templates** in the sidebar.
**Narration:** "The Templates library holds every page structure you've saved. To create one, build a page you're happy with in the editor and click 'Save as Template.' To use a template, click Create Page — enter a title and URL slug, and the system creates a new page with the exact same sections and buttons, fully editable and independent from the template. This is the fastest way to create pages for new locations."
**Subtitles:**
- Save a page structure as a Template once, reuse for new stores in one click.
- Create Page from Template: enter title and slug, sections and buttons are copied automatically.

## 4. Campaigns (1–2 min)
**Screen:** Click **Campaigns** in the sidebar.
**Narration:** "Campaigns let you track a marketing initiative — summer promotion, holiday offer — as one unified thing across multiple shortlinks and pages. Click Create Campaign and fill in the name, description, status, associated page, and date range. The table shows each campaign's status badge, linked page, date range, number of shortlinks, and total clicks rolled up from all of them. Deleting a campaign never breaks its shortlinks — they just lose the campaign label but keep working."
**Subtitles:**
- Track marketing campaigns as one unified thing across multiple shortlinks.
- Click count rolls up from all shortlinks in the campaign.
- Deleting a campaign never breaks its shortlinks.

## 5. UTM Builder (1 min)
**Screen:** Click **UTM Builder** in the sidebar.
**Narration:** "When you post a link on Instagram, Facebook, or a printed flyer, UTM parameters let you know which channel is actually driving clicks. Fill in the Destination URL, then Source, Medium, Campaign, Content, and Term — or pick a Location from the dropdown to auto-fill that field. The preview URL updates live as you type. Click Copy URL to use it directly, or click Create Shortlink plus QR to generate a trackable shortlink with a QR code right here. Important: changing a shortlink's destination does NOT change its QR code image — so you can update where it points without reprinting your flyers."
**Subtitles:**
- UTM parameters track which channel drives the most clicks.
- Preview URL updates live as you fill in the fields.
- Changing a shortlink destination does NOT change its QR code image.

## 7. Customer Service (1 min)
**Screen:** Click **Customer Service** in the sidebar.
**Narration:** "Customer Service lets you post a temporary Service Notice — for example 'Online ordering temporarily unavailable at Stone Oak' — displayed as a banner on the public Link Hub page. Set the name, message, type, and scope — global or per store — plus a start and end date so it expires automatically. Important: this screen can read notices from the server, but creating or editing notices from the Admin currently saves locally in your browser. Do not rely on this screen alone for urgent customer banners until the server-side save is fully connected. Always verify the notice appears on the public Link Hub after saving."
**Subtitles:**
- Post a temporary Service Notice as a banner on the public Link Hub.
- Set type, scope, and auto-expiry with start and end dates.
- IMPORTANT: Admin create/edit currently saves locally — verify on the public page before relying on it.
