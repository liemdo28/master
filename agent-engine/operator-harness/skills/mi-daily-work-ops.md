# mi-daily-work-ops

Use for daily work automation across calendar, email, files, Drive, Asana, dashboard, and website actions.

1. Classify each operation as read-only, reversible write, or high-impact action.
2. Apply the Approval Gate level before executing writes.
3. Require rollback or compensating-action notes for every write path.
4. Preserve audit trail entries with target, action, approval source, timestamp, and result.
5. Verify the owner-facing daily summary stays concise and action-oriented.
